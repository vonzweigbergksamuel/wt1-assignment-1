Fråga 3 - CSS

a. Vad är en regel i CSS och vad består den av?

En regel i CSS, även känt som stilregel, används för att definiera hur olika HTML-element ska se ut. En regel består utav två olika delar, en selektor och en deklaration. Selektorn används för att deklarera vilken typ av element eller vilket element som ska ändras. Här finns det framförallt tre olika typer av selektorer och de är elementselektorer, klasselektorer och id-selektorer.

b. Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Skillnaden mellan id och class är hur de används. Ett id används generellt bara för att ändra unika element och bör då bara användas en gång på en sida. En class används, till skillnad från ett id, för att stilsätta flera typer av element på en sida. Man kan med andra ord tilldela samma klass till flera HTML-element.
